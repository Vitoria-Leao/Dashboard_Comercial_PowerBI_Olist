# Dashboard_Comercial_PowerBI_Olist
Este repositório armazena o dashboard criado no Power BI utilizando os dados de e-commerce da Olist.

## Objetivo

Construir um dashboard comercial para monitorar indicadores estratégicos de vendas.

## Ferramentas
- Power BI;
- DAX.

## Dataset

Brazilian E-Commerce Public Dataset by Olist.

## KPIs
- Receita Total;
- Pedidos;
- Clientes;
- Ticket Médio;
- Crescimento.

## Páginas do Dashboard
- Visão Executiva;
- Produtos;
- Clientes;
- Geografia.

## Principais Insights

- A receita apresentou crescimento consistente entre 2017 e 2018;
- A categoria Beleza e Saúde representa a maior receita, mas não o maior número de pedidos;
- Os clientes não possuem mais do que uma compra na loja;
- São Paulo concentra a maior parte das vendas.

## Recomendações
- Criar campanhas para aumentar o número de pedidos nas categorias mais rentáveis;
- Desenvolver estratégias de fidelização de clientes;
- Expandir a atuação em estados com baixa penetração.
